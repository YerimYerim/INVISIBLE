# Player
 - 플레이어는 원으로 생겼으며 내부에 획득한 stone 들이 회전하고 있다. 
 - 가장 최근에 획득한 stone 의 색으로 변경된다.
 - 모든 색의 stone 을 획득해야지만 코어 색 하나를 부술수 있다. 
 - 가속도가 붙을 경우 붙은 속도에 따라 꼬리의 길이가 달라진다.
